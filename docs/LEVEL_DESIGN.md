# Shadow Clone - Level Design Plan

## Level Design Goals

- Teach the mechanic quickly
- Keep each room readable in one screen or near one screen
- Introduce one idea at a time
- Build difficulty through timing, not feature count

## Level Count

Target: 3 to 4 short levels maximum.

This keeps scope realistic and encourages polish over quantity.

## Level 1 - First Recording

Purpose:
Teach movement, recording, replay, and the idea that the clone can help solve a problem.

Setup:

- Small safe room
- Simple gap or timing gate
- One obvious clone use case

Success criteria:

- Player understands how to record
- Player sees the clone replay clearly
- Player completes the room in under a few minutes

## Level 2 - Button and Door

Purpose:
Teach cooperation between the live player and the replay clone.

Setup:

- One pressure button
- One linked door
- Player must record the clone standing on the button while they move through the door

Success criteria:

- The puzzle requires timing but stays readable
- Resetting is fast

## Level 3 - Hazard Timing

Purpose:
Combine clone replay with hazard awareness.

Setup:

- Hazard area
- Button or moving path that requires the clone to hold an interaction at the correct time
- More precise movement path

Success criteria:

- The room feels like a meaningful escalation
- Failure teaches timing instead of feeling random

## Level 4 - Final Chamber

Purpose:
Deliver a short final challenge using all prior ideas.

Setup:

- Button and door
- Hazard pressure
- Slightly longer solution path
- Clear visual goal

Success criteria:

- Feels like a final exam of the prototype
- Can still be completed in a short session

## Layout Guidelines

- Prefer strong silhouettes and clean collision shapes
- Avoid visual clutter
- Keep interactables easy to identify
- Use color coding for doors, buttons, hazards, and goal

## Difficulty Guidelines

- Level 1 teaches
- Level 2 confirms understanding
- Level 3 challenges timing
- Level 4 combines all learned ideas

## Scope Warnings

Avoid:

- Maze-like navigation
- Backtracking-heavy rooms
- Multiple simultaneous systems introduced at once
- Precision platforming that distracts from the clone mechanic
