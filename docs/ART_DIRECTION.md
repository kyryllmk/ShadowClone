# Shadow Clone - Art Direction

## Visual Goal

Create a minimalist sci-fi lab / simulation chamber style that feels clean, modern, and achievable with simple 2D assets.

The art direction should support gameplay readability first.

## Style Keywords

- Minimalist
- Futuristic
- Clinical
- High-contrast
- Mechanic-first

## Environment Direction

- Flat or lightly layered 2D shapes
- Clean wall panels and platform surfaces
- Subtle glow accents
- Geometric doors, buttons, and hazard zones

## Color Direction

Suggested palette approach:

- Dark neutral background
- Cool gray platforms
- Cyan or teal for interactive tech surfaces
- Red for hazards
- Green or cyan confirmation for active puzzle states
- White text for UI

Keep the palette limited and readable.

## Character Direction

Player:

- Simple silhouette
- Clear head/body contrast
- Easy to read in motion

Clone:

- Same shape family as player
- Distinct ghosted or tinted version
- Slight transparency or glow outline

## UI Direction

- Clean panels
- Clear typography
- Small amount of glow or line accents
- No heavy decorative elements

## Animation / Motion Direction

- Small squash or anticipation if easy
- Door open/close motion
- Button press feedback
- Clone spawn / vanish effect

These should remain subtle and low-cost.

## Production Guidance

- Start with primitive placeholders
- Lock gameplay before polishing visuals
- Upgrade only the most visible assets
- Reuse shapes and color rules across all levels
